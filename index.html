LiteEmbed: Adapting CLIP to Rare Classes

1 CVIT, IIIT Hyderabad
2 Adobe Research Bengaluru

Abstract

Large-scale vision-language models such as CLIP achieve strong zero-shot recognition but struggle with rare or under-represented classes. We introduce LiteEmbed, a lightweight framework that adapts CLIP to such classes using only a few reference images. LiteEmbed optimizes a learnable text embedding with image–text alignment while preserving the semantic structure of CLIP's embedding space using PCA-based subspace constraints. The resulting embeddings are plug-and-play and improve performance across classification, retrieval, segmentation, detection and generation tasks.

Method

LiteEmbed adapts CLIP to a new class by optimizing a learnable token embedding within the prompt “a photo of a *”. The embedding is trained using a small set of reference images. To prevent semantic drift and collapse, we guide optimization using coarse and fine subspace constraints derived from the geometry of CLIP’s embedding space.

Understanding CLIP Geometry

We analyze CLIP's text embedding space to understand how semantic concepts are organized. Naïve optimization can distort this structure and collapse embeddings. LiteEmbed preserves this geometry while enabling adaptation.

Results

LiteEmbed improves few-shot classification performance across multiple datasets and shows strong robustness in continual adaptation settings where new classes are introduced sequentially.

Plug-and-Play Across Tasks

Because LiteEmbed modifies only the text embedding, the learned representation can be directly used with existing CLIP-based pipelines. We demonstrate improvements across retrieval, segmentation, detection and generation.

Limitations

LiteEmbed relies on the semantic structure of CLIP’s embedding space. In cases of extreme visual ambiguity or complex compositional prompts, adaptation may still produce imperfect alignment.
